Sistema de Login em Python

Projeto desenvolvido em Python com o objetivo de praticar lógica de programação,
organização de código e conceitos básicos de sistemas de autenticação.

O sistema permite o cadastro e login de usuários, com persistência de dados em arquivo JSON.

Funcionalidades
- Cadastro de usuários
- Login com validação de usuário e senha
- Armazenamento de dados em arquivo JSON
- Operações básicas de CRUD na lógica do sistema

Interface Gráfica
O Tkinter foi utilizado apenas para as telas de login e cadastro.
As demais funcionalidades são realizadas diretamente na lógica do sistema,
mantendo o código mais simples e organizado.

## Tecnologias Utilizadas
- Python 3
- Tkinter
- JSON

## Como Executar
1. Certifique-se de ter o Python 3 instalado
2. Execute o arquivo `login.py` ou `sistema.py` pelo Python
3. Utilize a interface gráfica para realizar o login ou cadastro
